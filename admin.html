🔧 One-time Firebase setup

This app needs a free Firebase project to store GPS data. Follow these steps (takes ~3 minutes):

1

Go to console.firebase.google.com → Click Add project → name it salestrack-gps → Disable Google Analytics → Create project

2

In left sidebar: Build → Authentication → Get started → Enable Email/Password provider → Save

3

In left sidebar: Build → Firestore Database → Create database → Choose Production mode → Pick any region → Enable

4

In Firestore: click Rules tab → Paste this and Publish:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if request.auth != null;
    }
  }
}

5

Go to Project Settings (gear icon) → Scroll to Your apps → Click </> Web → Register app → Copy the config values below:

All data stays in your own Firebase account. Free Spark plan supports up to 1 GB storage and 50K reads/day — more than enough for a sales team.
